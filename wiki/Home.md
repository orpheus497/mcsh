# Learn C Shell Scripting

Welcome to the comprehensive guide on writing C Shell (csh / tcsh / mcsh) scripts!

This wiki is designed for beginners who have no prior experience with the C Shell, as well as for users transitioning from POSIX shells (like `bash` or `sh`). By the end of this tutorial, you will have a solid understanding of C Shell's logic, structure, and capabilities.
